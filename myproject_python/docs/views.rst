Views
=====

This module defines the views for the blog application.

.. automodule:: blog.views
   :members:

View Functions
--------------

*  `all_articles`: Displays all articles.
*  `my_articles`: Displays articles written by the logged-in user.
*  `article_detail`: Displays the details of a specific article.
*  `add_article`: Allows users to create a new article.
*  `edit_article`: Allows users to edit an existing article.
*  `delete_article`: Allows users to delete an existing article.
*  `signup`: Allows users to create a new account.

Class-based Views
----------------

*  `CustomLoginView`: Custom login view that displays a success message.
